Free Instant Website Deployment (3 Clicks)

Follow these simple steps to host NextBase as a living, public website that other families can open directly in their browser without any tech setup.

1

1. Download This Web File

Right-click anywhere on this website preview page and select "Save Page As...", or download this template code directly and name the file precisely: index.html

2

2. Drag & Drop to GitHub

Go to GitHub.com (create a free account if you don't have one). Click New Repository, name it nextbase, and check "Public". Drag your index.html file directly into the browser to upload it!

3

3. Activate the Live Link

In your new repository, go to Settings ➔ Pages. Under "Branch", select main (or master), click Save. In 30 seconds, your site is live at: https://yourusername.github.io/nextbase/

How this serves other families:

Once live, any other moving family can visit your link and use the interactive sliders and profile managers to update the NextBase dashboard for *their* specific destination. No coding required—they simply interact with the webpage you hosted!
